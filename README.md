# 项目简介

建立这个 repo 的初衷是，希望可以为广大程序员提供不同领域、不同层次的书籍阅读清单。

关于书籍信息组织与呈现的一些想法：

* 通过树状结构组织书籍。举个例子，`编程语言` 的节点下面会有 `C++`、`Python` 等子节点。如此一来，可以方便读者做特定领域书籍的检索。 
* 做一些特定领域的学习推荐计划，比如“在 beginner 阶段应该看什么书”，“在 intermediate 阶段应该看什么书”等等。
* 推荐书籍需要包含相关信息，这些信息包括：
	* 对读者知识背景的要求。
	* 书籍评价，如书籍所涵盖知识点的深度与广度。
	* 读者预计需要投入的时间。
